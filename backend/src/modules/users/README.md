# Users Module

## Purpose
مدیریت کاربران و دسترسی‌ها در Molido AI

## User Types

### Admin
- مدیریت کامل سیستم
- تنظیمات
- مشاهده همه گزارش‌ها

### Manager
- مدیریت فروشگاه یا شرکت
- مشاهده گزارش‌ها
- مدیریت کارکنان

### Seller
- ثبت فروش
- ایجاد فاکتور
- مشاهده کالاها

### Accountant
- حسابداری
- گزارش مالی
- ثبت هزینه‌ها

### Warehouse
- مدیریت موجودی
- ورود و خروج کالا

## Features

- ثبت نام کاربر
- ورود به سیستم
- مدیریت نقش‌ها
- سطح دسترسی
- امنیت حساب‌ها
