# Molido AI Database Design

## Products (کالاها)
اطلاعات کالا:
- Product ID
- نام کالا
- بارکد
- دسته‌بندی
- واحد اندازه‌گیری
- قیمت خرید
- قیمت فروش
- درصد سود
- تاریخ تولید
- تاریخ انقضا
- موجودی

## Invoices (فاکتورها)

انواع:
- خرید
- فروش

اطلاعات:
- شماره فاکتور
- تاریخ
- مشتری یا تأمین‌کننده
- لیست کالاها
- تعداد
- قیمت واحد
- تخفیف
- مالیات
- مبلغ نهایی
- وضعیت پرداخت

## Customers (مشتریان)
- شناسه مشتری
- نام
- شماره تماس
- آدرس
- حساب بدهکاری و بستانکاری

## Suppliers (تأمین‌کنندگان)
- شناسه تأمین‌کننده
- نام
- اطلاعات تماس
- سابقه خرید

## Inventory (انبار)
- کالا
- ورود
- خروج
- موجودی فعلی
- محل نگهداری
- هشدار کمبود

## Users (کاربران)
سطح دسترسی:
- مدیر
- فروشنده
- حسابدار
- انباردار

## AI Intelligence

هوش مصنوعی:
- تحلیل فروش
- پیش‌بینی موجودی
- پیشنهاد قیمت فروش
- هشدار تاریخ انقضا
- گزارش مدیریتی
