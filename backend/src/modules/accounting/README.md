## Financial Records

- درآمدها
- هزینه‌ها
- پرداخت‌ها
- دریافت‌ها

- صندوق
- حساب مشتریان
- حساب تأمین‌کنندگان

- گزارش هزینه
- سود خالص
- گردش مالی

- پیش‌بینی درآمد
- تشخیص هزینه‌های غیرعادی
- پیشنهاد کاهش هزینه
