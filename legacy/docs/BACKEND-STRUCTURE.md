# Molido AI Backend Structure

## Authentication
- ثبت نام
- ورود
- مدیریت کاربران
- سطح دسترسی

## Products Module
- کالاها
- دسته‌بندی‌ها
- قیمت‌ها
- تاریخ انقضا
- موجودی

## Sales Module
- فاکتور فروش
- مشتری
- پرداخت‌ها

## Purchase Module
- فاکتور خرید
- تأمین‌کننده
- ورود کالا به انبار

## Inventory Module
- ورود و خروج کالا
- کنترل موجودی
- هشدار کمبود

## Accounting Module
- درآمد
- هزینه
- سود و زیان

## AI Module
- تحلیل فروش
- پیشنهاد خرید
- هشدار هوشمند
