# Inventory Module

## Purpose
مدیریت موجودی و گردش کالا در Molido AI

## Inventory Features

- ثبت ورود کالا
- ثبت خروج کالا
- موجودی لحظه‌ای
- حداقل موجودی
- محل نگهداری کالا

## Stock Movement

- خرید
- فروش
- برگشت از فروش
- اصلاح موجودی

## Expiration Control

- بررسی تاریخ انقضا
- هشدار کالاهای نزدیک به انقضا
- گزارش کالاهای منقضی

## AI Features

- پیش‌بینی زمان تمام شدن کالا
- پیشنهاد سفارش خرید
- تشخیص کالاهای کم‌گردش
- جلوگیری از خواب سرمایه
