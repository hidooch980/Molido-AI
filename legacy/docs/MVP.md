# Molido AI MVP

## هدف نسخه اول
ساخت یک سیستم مدیریت فروشگاهی هوشمند تحت وب.

## امکانات اصلی

### کاربران
- ثبت نام
- ورود
- سطح دسترسی

### کالا
- ثبت کالا
- دسته‌بندی
- بارکد
- قیمت خرید
- قیمت فروش
- موجودی
- تاریخ انقضا

### خرید
- ثبت فاکتور خرید
- افزایش موجودی
- ثبت تأمین‌کننده

### فروش
- ثبت فاکتور فروش
- کاهش موجودی
- ثبت مشتری

### گزارش‌ها
- فروش روزانه
- سود فروش
- موجودی انبار

### هوش مصنوعی نسخه اول
- هشدار کمبود کالا
- هشدار تاریخ انقضا
- تحلیل ساده فروش
