# AI Module

## Purpose
هوش مصنوعی مرکزی Molido AI برای تحلیل و کمک به تصمیم‌گیری

## AI Features

## Sales Analysis
- تحلیل فروش
- شناسایی کالاهای پرفروش
- بررسی کاهش فروش

## Inventory Intelligence
- پیش‌بینی کمبود کالا
- پیشنهاد خرید
- هشدار تاریخ انقضا

## Financial Intelligence
- تحلیل سود و زیان
- تشخیص هزینه‌های غیرعادی
- پیش‌بینی درآمد

## Smart Assistant

دستیار مدیر:
- پاسخ به سوالات کسب‌وکار
- ارائه گزارش ساده
- پیشنهاد تصمیم بهتر

## Future Features

- پیش‌بینی بازار
- پیشنهاد قیمت هوشمند
- یادگیری از رفتار کسب‌وکار
