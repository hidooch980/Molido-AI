# Molido AI Technology Stack

## Frontend Web
- React / Next.js
- طراحی واکنش‌گرا برای کامپیوتر و تبلت

## Mobile
- Flutter
- پشتیبانی Android و iOS

## Backend
- Node.js یا Python
- ساخت API امن

## Database
- PostgreSQL
- ذخیره اطلاعات اصلی سیستم

## AI Engine
- اتصال به مدل‌های هوش مصنوعی
- تحلیل فروش
- پیشنهادها
- گزارش هوشمند

## Cloud
- سرور ابری
- پشتیبان‌گیری خودکار
- بروزرسانی آسان

## Security
- مدیریت دسترسی کاربران
- رمزنگاری اطلاعات
- ثبت فعالیت‌ها
