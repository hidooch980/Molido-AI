# Molido AI Permissions

## Roles

### Owner

Full access

### Admin

- Manage users
- Manage settings
- Manage business

### Manager

- View reports
- Manage sales
- Manage inventory

### Accountant

- Accounting
- Payments
- Financial reports

### Seller

- Create sales invoices
- View products
- Customer management

### Warehouse Staff

- Inventory operations
- Stock movement

## Permission Types

- Create
- Read
- Update
- Delete
- Approve

## Security

- Role Based Access Control
- Activity Log
- Permission History
