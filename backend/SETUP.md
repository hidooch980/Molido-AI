# Molido AI Backend Setup

## Purpose

Backend core for Molido AI platform.

## Requirements

- Node.js
- npm
- PostgreSQL
- Redis
- Git

## Architecture

Backend follows:

- Modular Architecture
- REST API
- Secure Authentication
- AI Integration Ready

## Main Modules

- Auth
- Users
- Products
- Inventory
- Invoices
- Accounting
- Reports
- AI

## Development Flow

1. Install dependencies
2. Configure environment
3. Run database
4. Start server
5. Test APIs
