# Invoices Module

## Purpose
مدیریت فاکتورهای خرید و فروش در Molido AI

## Invoice Types

### Purchase Invoice
فاکتور خرید:
- تأمین‌کننده
- تاریخ خرید
- کالاها
- تعداد
- قیمت واحد خرید
- تخفیف
- مالیات
- مبلغ نهایی

### Sales Invoice
فاکتور فروش:
- مشتری
- تاریخ فروش
- کالاها
- تعداد
- قیمت واحد فروش
- تخفیف
- مالیات
- مبلغ نهایی

## Features

- شماره فاکتور خودکار
- چاپ فاکتور
- ثبت پرداخت
- اتصال به انبار
- اتصال به حسابداری

## AI Features

- بررسی قیمت غیرعادی
- محاسبه سود واقعی
- پیشنهاد قیمت فروش
- تحلیل فروش
